Meta Analysis ToolsÂ¶

These tools can be used to get information about your projects, event collections, Saved Queries, etc.

  • Versions Resource - The entry point to our APIs. Returns the available API versions and links to the Discovery Resource.
  • Discovery Resource - Returns the available child resources. Currently, the only child resource is the Projects Resource.
  • Projects Resource - Returns the projects accessible to the API user, as well as links to the Project Row Resource for discovery.
  • Project Row Resource - Returns detailed information about the specific project, as well as links to related resources.
  • Event Collections List Resource - Returns detailed information about all the event collections in the specific project, as well as links to them. Also used for bulk-inserting events.
  • Event Collection Row Resource - Returns event collection info including properties and their type. It also returns links to sub-resources. Also used for inserting events.
  • Queries Resource - Returns links to the available Queries, i.e. Metrics, Progressions, Data Extractions.
  • Saved Queries List Resource - Returns all the available Saved Queries & Queries in this project as well as links to each of their Saved Query Row Resource.
