Keen.io helps developers understand how people use the applications they create. We provide cloud infrastructure and REST APIs for Data Collection and Data Analysis. Use Keen.io to record user interactions, then use our APIs to analyze your data. One more thing - we also support full data extraction so you can get your data into Excel, Hadoop, or any other tools you might need.

Choose one of these options to get started:

  • Our Getting Started Guide will walk you through a hands-on cURL example.
  • iOS devs: jump straight into integrating your iOS app. Download our SDK.
  • Ruby folks: Gem install Keen. Reference our Ruby Client Usage Guide.
